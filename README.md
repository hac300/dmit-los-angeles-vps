## DMIT 洛杉矶三条线路，差别比你想的大

DMIT 洛杉矶（LAX）机房目前开放三条线路：Premium（Pro）、Eyeball（EB）和 Tier 1（T1）。名字有点迷，但逻辑其实很清晰。

**Premium / CN2 GIA 三网优化**

这是旗舰线路。电信去程走 AS4809（CN2 GIA），联通去程直连 AS4837，移动去程绕香港 CMI，三网回程统一走 CN2 GIA 回来。实测晚高峰三网平均延迟在 155-180ms 左右，丢包率基本在 0.1% 以下。

这条线路的核心价值是：不管你在国内哪个运营商，回程都走 CN2 GIA，不会因为是移动用户就被绕到不知道哪里去。对电信用户来说效果最明显，联通次之，移动也能接受。

**Eyeball / CMIN2 + 9929 回程**

次旗舰。电信联通去程走 CN2/AS9929，移动走香港 CMI，但回程三网统一走 CMIN2（AS58807）和 9929 负载均衡。

说实话，对联通和移动用户来说，Eyeball 的性价比反而更高——走 CMIN2 对这两个运营商的覆盖更直接，而价格比 Pro 低一档。如果你不是电信用户，Eyeball 值得优先考虑。

**Tier 1 / 国际 BGP 线路**

纯国际线路，没有大陆专项优化。跑的是全球 Tier 1 运营商 BGP 路由，国际互联质量很好，但对中国大陆用户来说延迟会偏高。适合做国际站、CDN 源站、科学上网落地节点等不特别依赖大陆访问速度的场景。价格是三条线里最低的。

不要买错：如果你的核心用户在国内，Tier 1 不是合适选择。电信用户尤其要注意——买了 T1 之后因为绕路，延迟可能比洛杉矶到国内正常情况还高。
